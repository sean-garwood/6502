# Building a 6502 Computer

This project follows along with Ben Eater's course on [building a 6502
computer](https://eater.net/start).

## Project structure

```
.
├── bin              # executable code
├── datasheets
│   ├── 6502
│   └── clock_module
├── resources        # links, etc
├── schematics
│   ├── 6502
│   └── clock_module
└── src              # source code
```
